# URL Shortener Microservice

## Short URL Creation

### APIs and Microservices Projects for FCC

Example:

  POST [project_url]/api/shorturl - https://www.google.com 

Example Usage:

    [project_url]/api/shorturl/_xbhS1xqP

Will Redirect to:

    https://www.google.com
